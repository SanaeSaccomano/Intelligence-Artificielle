# ![little_icon](https://user-images.githubusercontent.com/91669852/178104798-ce6481da-a9dd-4b29-99e0-38eb6f6f106d.png) Création d'une application Language Understanding - LUIS
## Description

Projet portant sur la création, l'évaluation et l'amélioration d'une solution Language Understanding d'Azure de Microsoft

L'analyse de la base de données se trouve dans le notebook [Analyse frames](https://github.com/SanaeSaccomano/Intelligence-Artificielle/blob/main/Application%20LUIS%20Azure/Analyse%20frames.ipynb)

La création d'une première application Language Understanding d'Azure de Microsoft et son évaluation se trouve dans le notebook [Creation Application LUIS]

Une proposition d'amélioration de l'application LUIS et de son évaluation se trouve dans le notebook [Upgrade Application LUIS](https://github.com/SanaeSaccomano/Intelligence-Artificielle/blob/main/Application%20LUIS%20Azure/Upgrade%20Application%20LUIS.ipynb)

Un coup d'oeil au projet se trouve au sein de la présentation [Application LUIS](https://github.com/SanaeSaccomano/Intelligence-Artificielle/blob/main/Application%20LUIS%20Azure/Application%20LUIS.pdf)
 
## Compétences
- Exploration et préparation des données
- Création évaluation d'une application LUIS grâce à des scripts create/train/test/predict
- Utiliser une application LUIS de façon sécurisé

## Librairies

- numpy

- json, time, uuid, urllib, zipfile, os
- azure.cognitiveservices.language.luis
- msrest.authentication
